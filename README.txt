To run the project, open the AIML Project in the terminal of VS Code and type 'python model.py'

THat is the first part of the code that allows us to determine which of the four models work the best with our dataset.

Next, run the app.py by typing 'python app.py' The Flask app should run on a port, number bring displayed on the terminal. Go to the port specified on a browser to see the front end of the project and interact with the model to give it custom values for the test data.
